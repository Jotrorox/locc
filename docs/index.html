LOCC

Lines of Code Counter
Features Install Usage GitHub

Fast & Beautiful Code Counting

A modern command-line tool that counts lines of code across 90+ programming languages with colorful terminal output.

Get Started Download v0.1.0
Terminal
$ locc ./my-project
Scanning: ./my-project
──────────────────────────────────────
Go 5 files 234 lines
JavaScript 3 files 156 lines
TypeScript 2 files 89 lines
──────────────────────────────────────
Total 10 files 479 lines

Features

Lightning Fast

Optimized scanning with efficient file processing and large buffer sizes for maximum performance.

Beautiful Output

Colorful terminal output powered by Charm's Lipgloss library for an amazing user experience.

Dual Modes

View results grouped by language type or see individual files with detailed line counts.

30+ Languages

Supports all major programming languages including Go, JavaScript, Python, Rust, and many more.

Smart Counting

Excludes empty lines and whitespace-only lines for accurate code metrics.

Cross-Platform

Works seamlessly on Linux, macOS, and Windows with pre-built binaries available.

Installation

Download Binaries

Get the latest pre-built binaries for your platform:

Linux x86_64 Linux ARM64 Windows x86_64 Windows ARM64

Go Install

go install github.com/jotrorox/locc@latest

From Source

git clone https://github.com/jotrorox/locc.git
cd locc
go build -o locc main.go

Usage

Basic Usage

locc /path/to/your/project

File Mode

locc --file-mode /path/to/your/project
# or
locc -f /path/to/your/project
